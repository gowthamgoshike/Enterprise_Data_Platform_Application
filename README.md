Enterprise ML Data Platform (AWS Free Tier)
Overview

This project implements a cloud-native Enterprise Data Platform with an integrated MLOps pipeline using AWS Free Tier services.

Architecture

Data Source → S3 Data Lake → Spark ETL → RDS PostgreSQL → MLflow → FastAPI → Streamlit Dashboard → Docker → EC2
